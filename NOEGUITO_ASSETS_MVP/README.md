# NOEGUITO_ASSETS_MVP

Pack MVP generado a partir del diseño elegido de Noeguito.

## Qué incluye
- Hoja maestra completa.
- Personaje principal.
- Vistas turnaround: frontal, 3/4, perfil y espalda.
- Expresiones básicas.
- Bocas básicas.
- Poses base.
- Taza y vapor.

## Importante
Este pack está generado desde la imagen final del diseño, por lo que son PNGs recortados con fondo transparente aproximado.
Sirve para empezar en Canva, CapCut, Premiere, After Effects o como referencia para un ilustrador.

No es todavía un rig profesional por capas reales.
Para animación pro habría que redibujar Noeguito en vector y separar cabeza, brazos, manos, ojos, bocas, gorra, hojas, delantal, taza y vapor como capas independientes.

## Recomendación
Usad este pack como MVP visual. 
Cuando el diseño quede cerrado al 100%, se crea el pack PRO en AI/SVG/PSD por capas.
